Maze Bot

About This App

Welcome to Maze Bot,
With our desktop application you can create mazes of the size you like, and export them in a png file.
Maze Bot uses Depth First Search algorithm to generate a random maze.

Why this project?

This is a project created for our class of Operating System (CSI - 345) by the Windows Team
The goal was to creat an application for Windows Operating System.
We used Windows Presentation Foundation's framework to develop this application.
